# docs/

Documentation generated by selfdoc. Also contains the TOML format reference.

## selfdoc

selfdoc.json at project root points here. `selfdoc gen` produces API reference documentation for all internal packages. Deployed to Cloudflare Pages at pgdesign.smmh.dev.

## Manual docs (templates)

- `docs/_README.md` -- selfdoc template for root README
- `docs/format-reference.md` -- Complete TOML schema format specification
- `docs/semantic-types.md` -- Builtin and user-defined type documentation
- `docs/validation-rules.md` -- All E/W codes with examples
- `docs/migration-guide.md` -- How migrations work, format, lifecycle
- `docs/quickstart.md` -- Getting started (introspect existing DB or create from scratch)
